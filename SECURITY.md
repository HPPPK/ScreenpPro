# 安全策略

## 安全边界

ScreenPro 的覆盖式屏保是应用级全屏展示工具，不是操作系统锁屏。它不能可靠阻止 `Alt + Tab`、任务管理器、系统安全快捷键、管理员操作、强制退出或物理访问攻击。需要真正的安全锁定时，Windows 用户应使用应用内“安全锁定 Windows”按钮；该操作调用 Windows 原生锁屏并由系统账户认证。

## 报告安全问题

请不要在公开 Issue 中披露密码处理、签名密钥、更新验证或本地文件访问方面的漏洞。请通过仓库拥有者的 GitHub 账户私信联系，并提供：

- 复现步骤；
- 影响范围；
- 可能的修复建议（如有）。

## 密钥处理

Tauri 更新私钥仅可保存在本机安全位置和 GitHub Actions Secret 中；不得提交到仓库。
